How to use this form

Completing the certificate

  1. Fill in the Customer Name and Customer Facility Address at the top.
  2. In Section 2, fill in the information for the equipment involved in the service.
  3. In Section 3, fill in the details on the service type and time frame.
  4. In Section 4, fill in the details on the parts used during service. If item was requested via Monday, include the item ID for the request in the notes section.
  5. Section 5, record any deviations or non-conformances (e.g. NCM, FIT, Jira engineering ticket, etc.).
  6. Section 6, record the summary of the work performed. Summary should have sufficient detail that another FSE can read and understand the scope of work performed.
  7. If applicable, in Section 7, fill in the details on the preventative maintenance service.
  8. Attach photos and supporting documents in applicable Sections. Drag-and-drop or click to browse. Photos and documents render at the end of the printed PDF, one per page.
  9. Have an authorized supplier representative sign Section 8, then press Create PDF to print the controlled record. If no Customer Representive is available, check the "Customer unavailable" box and email the signed Service report to the customer.

What the buttons do

Save Progress Downloads a draft HTML file with everything you've entered so far. Reopen that file later to resume — entries, attachments, and signature are restored.
Clear All Resets the entire form (asks for confirmation first). Use only if you want to start over.
Create PDF Saves an HTML copy of the as-submitted record and opens the print dialog so you can save the controlled PDF. Photos and documents are appended after the form.
⚠ Controlled record
The PDF is the controlled record. When completing the Field Service Report only the signed PDF produced by Create PDF shall be provided to the customer and stored by Ultima Genomics. The HTML files (both _DRAFT.html and the as-submitted .html) are working copies and can be discarded after the PDF is created.
